    Python IDE for Mac

    Introduction

    Liasis is a Python IDE for Mac. Written with Cocoa, it feels right at home with your other Mac apps, while providing the tools you need for efficient Python development.

    Features

    Code editor

    • Code minimap
    • Tabbed file organization
    • Syntax coloring
    • Automatic formatting
    • File browser
    • Code introspection
      • Autocomplete
      • Variable highlighting
      • Code navigation

    Built-in interpreter

    • Execute Python code directly within Liasis

    Installation

    Liasis requires Mac OS 10.8 and higher. Download the latest stable build here.

    The development version of Liasis is kept in the develop branch of each repository (Liasis, the LiasisKit framework, and the plugins). After building each, follow the installation instructions for the framework and plugins found in the README files.

    Future plans

    Liasis is under active development. We plan to introduce the following features:

    • Projects
    • Split views and saved workspaces
    • Code linting
    • Project documentation
